puma-project
============

This is the top-level project bundling all the sub-modules of PUMA.

To clone everything, run:

```
git clone https://github.com/PUMA-IAM/puma-project
git clone https://github.com/PUMA-IAM/puma-pip-utils
git clone https://github.com/PUMA-IAM/puma-pep-utils
git clone https://github.com/PUMA-IAM/puma-util
git clone https://github.com/PUMA-IAM/puma-central-pdp-rmi-utils
git clone https://github.com/PUMA-IAM/puma-sunxacml
git clone https://github.com/PUMA-IAM/puma-central-pdp
git clone https://github.com/PUMA-IAM/puma-application-pdp
git clone https://github.com/PUMA-IAM/puma-service-mgmt
git clone https://github.com/PUMA-IAM/puma-service-mgmt-tenant
git clone https://github.com/PUMA-IAM/puma-service-mgmt-provider
git clone https://github.com/PUMA-IAM/puma-service-authentication
git clone https://github.com/PUMA-IAM/puma-service-authentication
git clone https://github.com/PUMA-IAM/puma-application
```
